# Place Your Figures Here

Put all generated and exported figures in the subfolders below.
Scripts auto-save outputs here. You can also add any manually exported plots.

```
figures/
  phase1/     ← CLR cluster plots (auto-generated by phase1_aitchison.py)
  phase2/     ← Neural ODE trajectory plots (auto-generated by phase2_neural_ode.py)
  phase3/     ← MCMC trace and posterior plots (auto-generated by phase3_mcmc.py)
  phase4/     ← R_idx heatmaps and trajectory plots (auto-generated by phase4_robustness.py)
```

Any figures exported from other tools (e.g. from the Word document, presentations) can go in the
relevant subfolder. Name them clearly:

Examples:
  phase3/ridx_posterior_HbC.png
  phase4/heatmap_balancing.png
  phase2/ensemble_panaxia.png
